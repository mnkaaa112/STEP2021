## 宿題1
### 行列積とメモリアクセス

for文の順序による実行時間の予想  
|for文の順序|c[i][j]|a[i][k]|b[k][j]|
|----|----|----|----|
|i-j-k|不変|連続|**不連続**|
|i-k-j|連続|不変|連続|
|j-i-k|不変|連続|**不連続**|
|j-k-i|**不連続**|**不連続**|不変|
|k-i-j|連続|不変|連続|
|k-j-i|**不連続**|**不連続**|不変|

したがって、実行速度は  
i-k-j = k-i-j > i-j-k = j-i-k > j-k-i = k-j-i の順で速いと予想される
